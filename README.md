# NBook

A site to help you plan your reading journey

*Having trouble trying to keep track of what you planned to read next? Unsure when to order a book now so it will be here by the time you're ready to start on it? Simply add your books to this todolist. Using page counts and estimated daily reading, you can project when you will finish with your book. Or just use it as a journal for what you've read.*

## Getting Started

Simply clone this repo
```
git clone https://github.com/nathanctay/nbook.git
```
and install the dependencies
```
npm install
```
Then you're ready to go
